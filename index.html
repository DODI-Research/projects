Code Demo

Auto-Foley is a proof-of-concept workflow that automates sound design for silent videos, using a vision language model for scene analysis.

Overview

The workflow combines a vision language model to analyze scenes and generate text prompts, with a text-to-sfx model to create appropriate sound effects. These sounds are then composited back onto the original video at their appropriate timings.
